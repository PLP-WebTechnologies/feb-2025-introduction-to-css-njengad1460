/* types of selector:
    1.universor selector (*)
    This apply to all elements
    2. Element selector
    target element of a specific type 
    ie table, p...
    3. class selector
    target element with specific class atributes 
    ie <h3 class= "one" ></h3>
    <p class ="one" ></P>
    
    Targeting this we start with a dot and then class-name
    
    .one{...
    }
    4. ID selector
    this targets a specific element (#) we use that symbol 
    ie <ol id="list"></ol>

    #list{............
    }
    5. Group selectors
    this target multiple selectors
    ie p,h3,h6....{....
    }

    */
